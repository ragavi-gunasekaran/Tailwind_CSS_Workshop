# Karlskrona Tourism Website

## Overview

This is a Karlskrona Tourism theme web page which is build based on Tailwind CSS components.

## Pages

- home.html
- places.html
- activities.html
- packages.html
- news.html
- contact.html

## Technologies Used

- HTML5
- Tailwind CSS CDN

## How to Run

Open `home.html` in the browser.
